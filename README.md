New or WIP packs should be put in their own modpack folder.

Tested and functioning packs can be merged into ninitale.Ninitale.

## Installing

**Before adding to the server:**
1. Restart the server to save everything and kick everyone.
2. Wait for it to boot back up fully.
3. Back up everything on the server except the Server/ folder and Assets.zip file.

**Install:**
1. Place the pack folders in the server's "mods/" folder, e.g. "mods/ninitale.Ninitale/"
2. Restart server again.
